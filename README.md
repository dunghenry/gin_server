### Create

```js
go mod init username/name_project
```

### Install Gin

```js
go get -u github.com/gin-gonic/gin
```

### Import net/http

```js
import "net/http";
```

### Install mongo-driver

```js
go get -u go.mongodb.org/mongo-driver/mongo
```

### Delete all package not used

```js
go mod tidy
```
